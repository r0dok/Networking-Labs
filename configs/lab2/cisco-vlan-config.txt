 no shutdown

! Trunk to Raspberry Pi
interface fastEthernet 0/2
 description Trunk to Raspberry Pi Router
 switchport mode trunk
 switchport trunk allowed vlan all
 switchport trunk native vlan 10
 no shutdown

interface fastEthernet 0/10
